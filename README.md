# Les Ailes

## Project Description

Les Ailes is a web development project using Pug templates. This project aims to create a dynamic and responsive website.

## Features

- Dynamic content rendering with Pug
- Responsive design
- Easy to maintain and extend

## Tech Stack

- **Pug**: Template engine for Node.js
- **HTML**: Markup language for creating web pages
- **Sass**: CSS preprocessor for more efficient styling
- **CSS**: Style sheet language for designing web pages
- **JavaScript**: Programming language for web development
  **LiveScript**: A language that compiles to JavaScript, offering a more expressive syntax and additional features

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/B1lol-dev/Les-Ailes-Simple-Pug-Clone.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Les-Ailes-Simple-Pug-Clone
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
